Instructions before you upload

  1. Prepare Your CSV File

    1. Ensure that your CSV file contains headers for each column.

  2. Set Up Your Padlet Board

    1. Update settings
      • Go to Settings -> Heading -> Layout
      • Set the format to "Wall"
      • Enable "Group posts by section" option
    2. Create sections
      • e.g., if you have a header called "Fruits," there must be a section named "Fruits" on the board
      • Only create sections for the data you want to import

  3. Upload via Padlet Importer

    1. Complete the form
    2. Upload your CSV file(s).

  4. Check the Board

    • Verify the content is correctly placed in the right sections.
